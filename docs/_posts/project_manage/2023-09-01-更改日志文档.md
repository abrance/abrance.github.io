## 更改日志 - 主要石犀总控项目

此文档记录了主要石犀总控项目的主要更改，包括已完成的任务和已解决的Bug。每次主分支有新的代码提交时，请更新此文档。

### 文档规则：

1. 请按时间顺序记录，最新的更改放在顶部。
2. 为了保持格式统一，请按照以下模板进行填写。
3. 只记录合并到分支 release-2.2.0 的更改。
4. 包括提交的哈希值，以方便其他团队成员查看具体的代码更改。

---

### 更改日志模板：

```
### [日期]

**提交者**：[开发者姓名]

**提交哈希**：[Git提交的哈希值] 可以为列表

**已完成的任务**：

- [Task #ID] 简短描述。例如：
    - [Task #1234] 新增登录功能。

**已解决的Bug**：

- [Bug #ID] 简短描述。例如：
    - [Bug #5678] 修复了密码重置功能的问题。

---

```
