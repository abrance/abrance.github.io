typora插件.md

2023/11/1 resource Markdown plugin

插件概述

安装

github typora plugin use

使用

使用

常用插件快捷键

  • C-S-B 中英间隔 1 space
  • C-j 搜索
  • 鼠标右键 插件功能

常用插件功能

  • 文件模板
  • kanban 功能
  • 标签页（记得只保留一个，多了会出现现实问题）
  • markdown 格式检查

插件开关配置

  • linux
    • 右键 -> 非常用插件 -> 右键菜单 -> 打开插件配置文件
    • settings.default.toml 里的配置 拷贝 到 settings.user.toml 中，将对应插件的 （sudo gedit settings.user.toml）启用插件 ENABLE = false/true
    • （建议关闭标签页管理插件window_tab，经常有问题）
  • windows
