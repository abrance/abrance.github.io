研发JD

职位：Golang资深研发工程师

主要职责：

  1. 使用Go语言进行高效、可靠的软件开发。
  2. 设计、开发并维护复杂的软件系统和架构。
  3. 与团队成员合作，共同解决复杂的技术问题。
  4. 对数据库的使用和维护负责，确保性能和稳定性。
  5. 参与代码审查，确保代码质量和遵守最佳实践。
  6. 持续优化应用程序的性能，提高系统的可扩展性和可靠性。
  7. 编写清晰、可维护的代码，并对软件进行全面测试。

必备技能和资格：

  1. 具有4年以上使用Go语言的开发经验。
  2. 熟悉数据库的设计、优化和维护，包括但不限于MySQL、PostgreSQL或NoSQL数据库。
  3. 良好的团队合作能力和沟通技巧。
  4. 对新技术充满好奇心，并愿意不断学习和进步。
  5. 熟悉软件开发的最佳实践和敏捷开发方法。

加分项：

  • 有网络安全领域系统开发经验。
  • 熟悉 Devops 领域。
  • 熟悉信创平台、有相关研发经历。

职位：Golang中级研发工程师

主要职责：

  1. 使用Go语言进行有效、可靠的软件开发。
  2. 参与设计和开发大型复杂度的软件系统和架构。
  3. 与团队成员合作，解决技术难题。
  4. 协助数据库的使用和维护工作，确保性能和稳定性。
  5. 参与代码审查，确保代码质量。
  6. 协助优化应用程序性能，提升系统的可扩展性和可靠性。
  7. 编写清晰的代码和相关自测。

必备技能和资格：

  1. 具有2年以上使用Go语言的开发经验。
  2. 对数据库的基本使用和维护有一定了解，熟悉MySQL、PostgreSQL或NoSQL数据库。
  3. 具备良好的团队合作和沟通能力。
  4. 对技术保持好奇心，并愿意学习新知识。
  5. 了解软件开发的基本实践和敏捷开发流程。

加分项：

  • 有网络安全领域系统开发经验。
  • 熟悉 Devops 领域。
  • 熟悉信创平台、有相关研发经历。
