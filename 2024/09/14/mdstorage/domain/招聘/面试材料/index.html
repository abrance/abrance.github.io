自我介绍

我叫邓天翔，湖北人，毕业于河南工业大学，大学毕业后到深圳这边工作，最初在一家生物公司，它是做人类基因罕见病治疗诊断的，我负责内部的生产力平台，对接高通量的基因组测序，配合进行疾病的诊断流程，用的技术栈是 Python Django Celery RabbitMQ Numpy。
后来一家公司是做基础设施交付的，主要是超融合、存储技术，我负责医疗云系统、档案云系统的研发和维护工作。技术栈是 Python Golang JS KVM Ceph 区块链。
目前这个公司是做网络安全的，那时新来了一个技术总监，我来公司第一个项目是堡垒机项目，用了两周时间，将堡垒机的后端逻辑交付了第一个版本，当时就让我提前转正了。让我开始看云原生、高并发、流量治理的书，把我从堡垒机项目挖出来了，然后专门做这个流量治理项目，当时还叫 X 项目，我第一周，就把技术总监准备让我一个月搭出来的框架给搭好了，还做了文档和自动化部署，然后得到技术总监的表彰，就直接给我加薪了。那一年，总监和我招进来另外3个人，组成了这个项目的基石。6月后，正式发布了 1.0 版本，交付了 X 项目的引擎，然后总监走了，后面我成了项目的负责人，负责带领团队继续向前走。现在因为个人原因，想要尝试外面的机会。
