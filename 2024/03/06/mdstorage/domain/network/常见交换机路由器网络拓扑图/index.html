graph TD;
    subgraph WAN
    Internet[互联网]
    end

    subgraph LAN
    Router[路由器] --> |内网接口| Switch1[交换机1]
    Router --> |内网接口| Switch2[交换机2]
    Switch1 --> PC1[PC 1]
    Switch1 --> PC2[PC 2]
    Switch2 --> Terminal1[终端设备 1]
    Switch2 --> Terminal2[终端设备 2]
    Switch1 --> Switch3[交换机3]
    Switch3 --> PC3[PC 3]
    Switch3 --> Terminal3[终端设备 3]
    end

    Internet --> |外网接口| Router
