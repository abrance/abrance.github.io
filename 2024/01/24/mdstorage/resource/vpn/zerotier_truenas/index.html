truenas zerotier app使用

版本

truenas 23.10

zerotier app 1.0.9

NAS系统配置

  • 系统设置 -> 高级 -> 系统控制 -> 添加两个变量和值

配置

网络： 粘贴 zerotier 网站中的 Network ID

Network Configuration -> Host Network： 勾选

zerotier 配置

  • 登入 zerotier
  • 勾选刚刚接入的设备，复制分配给此设备的 $IP
  • Add Routes: 填入接入的网段，via： $IP

即可完成通过此 app，访问接入网段的目的

resource > vpn
